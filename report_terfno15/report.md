## 与えられた微分方程式をルンゲ・クッタ４次公式によって解くプログラムを作れ．
### (1)
$\frac{dy}{dx}=x+y$ を解く
runge-kutta.cpp
```
#include <stdio.h>

double f(double x, double y){
    return (x + y);
}

int main(){
    int i;
    double x, y, h, k1, k2, k3, k4, k;
    char zz;

    printf("\n\n4次のルンゲクッタ法により dy/dx=x+y を解きます。\n\n");
    printf("Returnキーを押してください。\n");
    scanf("%c",&zz);

    printf("  x                 y\n");

    x = 0.0;
    y = 1.0;
    h = 0.1; //刻み幅は0.1

    printf("%10.6lf %10.lf\n",x,y);

    for(i=1;i<=40;i++){
        k1 = h * f(x, y);
        k2 = h * f(x + (h / 2), y + (k1 / 2));
        k3 = h * f(x + (h / 2) , y + (k2 / 2));
        k4 = h * f(x + h, y + k3);
        k = (k1 + (2 * k2) + (2 * k3) + k4) / 6;
        x = x + h;
        y = y + k;
        
        printf("%10.6lf %10.lf\n",x,y);
    }
    return 0;
}

```
